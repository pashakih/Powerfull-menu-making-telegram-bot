# MultiChef Telegram Bot — Полные инструкции

Этот репозиторий содержит Telegram-бота для управления меню поваров и оформления заказов. Бот использует SQLite для хранения данных.
Измините файл для ваших целий.

Установка зависимостей

```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

Настройка токена

1. Скопируйте `.env.example` в `.env`:

```bash
cp .env.example .env
```

2. Вставьте ваш токен в `.env` (переменная `TELEGRAM_TOKEN`).

Запуск

```bash
# Рекомендуется запускать `bot.py` (скопировано из `bot.ini`)
python bot.py
```

Примечания по безопасности и продакшну

- Не храните токен в публичном репозитории.
- Для постоянной работы используйте systemd, Docker или облачный хостинг.
- Этот код подходит как PoC и для небольших проектов; для продакшна рекомендую добавить аутентификацию, валидацию входных данных и резервное копирование БД.
