# MultiChef Telegram Bot

Коротко: простой Telegram-бот для управления меню поваров и оформления заказов (SQLite).

Подготовка и запуск

1. Создайте виртуальное окружение и установите зависимости:

```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

2. Создайте файл `.env` на основе `.env.example` и вставьте ваш токен:

```bash
cp .env.example .env
# Отредактируйте .env и вставьте реальный токен
```

3. Запустите бота:

```bash
# Рекомендуется запускать `bot.py` (скопировано из `bot.ini`)
python bot.py
# или
nohup python bot.py > bot.log 2>&1 &
```

Примечания

- Файл `bot.py` — основной скрипт; он читает `TELEGRAM_TOKEN` из переменных окружения.
- Для постоянного размещения используйте systemd, Docker или облачные сервисы.
- Не храните реальный токен в публичных репозиториях. Используйте `.env` или секреты CI.
